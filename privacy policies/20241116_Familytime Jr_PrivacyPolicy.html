FAMILYTIME Jr. for ANDROID

FamilyTime Jr. (for childs android device) collects and provides you access to certain cell phone and online activities (including, but not limited to, call history, text messages, contacts, installed apps, location history and apps usage stats) from your childs phone or tablet on which the FamilyTime software is installed (the Accessed Activities). Data collected through use of the Service and is accessible to the FamilyTime Dashboard.

Accessibility Permission Required

FamilyTime Jr. requires Accessibility permission to work its main parental control features like ScreenTime, App Usage, App Blocker, Daily Limit, Web history, Youtube history, and Tiktok history.

Data Collection

FamilyTime Jr. collects the following data via accessibility API and uses it for its key features.

  • It collects the web history of the user

  • It gets the watch list of Tiktok and youtube videos

  • It collects all app (package) names to work ScreenTime, App Usage, App Blocker, Daily Limit

Note: FamilyTime Jr. App does not collect any financial or personal information and does not share and sell its data to third parties.
